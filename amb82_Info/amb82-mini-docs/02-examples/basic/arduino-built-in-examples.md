# Arduino Built-in Example

## AMB82-mini (RTL8735B) Supported Arduino Built-in Example Table

There are many built-in examples in Arduino. All examples that listed are compatible with Ameba.

- [Arduino built-in example details](https://www.arduino.cc/en/Tutorial/BuiltInExamples/)

### Basics

| Example | Comments |
|---------|----------|
| AnalogReadSerial | Connect potentiometer to 3.3V. ADC pin used, up to 3.3V. ADC pin options: A0-A7 |
| BareMinimum | - |
| Blink | LED_BUILTIN sets to LED_B (on-board LED blue). LED options: LED_B (blue) or LED_G (green) |
| DigitalReadSerial | - |
| Fade | For analogWrite(), use PWM capable pin (1-8, 23) |
| ReadAnalogVoltage | ADC pin used, up to 3.3V |
