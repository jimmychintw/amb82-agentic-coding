# BMI270 Wrist Gestures[](#bmi270-wrist-gestures)
## Materials[](#materials)
- [AMB82-mini](https://www.amebaiot.com/en/where-to-buy-link/#buy_amb82_mini) x 1
- [SparkFun 6oF IMU Breakout - BMI270](https://www.sparkfun.com/sparkfun-6dof-imu-breakout-bmi270-qwiic.html) x 1
## Example[](#example)
### Introduction[](#introduction)
This example shows how to enable the step detector, counter, and activity recognition features of the BMI270.
### Procedure[](#procedure)
Connect the AMB82-mini to I2C_SDA, I2C_SCL, INT1 of the sensor as shown in the diagram below.
[](../../../../_images/image01147.png)
Open the example in File -> Examples -> AmebaWire -> BMI270 -> WristGestures
[](../../../../_images/image02142.png)
Compile and run the example.
The result will be printed out upon interrupt trigger.
[](../../../../_images/image03121.png)
