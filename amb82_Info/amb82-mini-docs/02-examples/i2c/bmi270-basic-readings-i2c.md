# BMI270 Basic Readings I2C[](#bmi270-basic-readings-i2c)
## Materials[](#materials)
- [AMB82-mini](https://www.amebaiot.com/en/where-to-buy-link/#buy_amb82_mini) x 1
- [SparkFun 6oF IMU Breakout - BMI270](https://www.sparkfun.com/sparkfun-6dof-imu-breakout-bmi270-qwiic.html) x 1
## Example[](#example)
### Introduction[](#introduction)
This example shows how to use SparkFun 6DoF BMI270 to get basic measurements using I2C.
### Procedure[](#procedure)
Connect the AMB82-mini to I2C_SDA and I2C_SCL of the sensor as shown in the diagram below.
[](../../../../_images/image01136.png)
Open the example in File -> Examples -> AmebaWire -> BMI270 -> BasicReadingsI2C
[](../../../../_images/image02131.png)
Compile and run the example.
The measurements from the sensor will be printed at 20ms interval.
[](../../../../_images/image03110.png)
