# BMI270 Interrupts[](#bmi270-interrupts)
## Materials[](#materials)
- [AMB82-mini](https://www.amebaiot.com/en/where-to-buy-link/#buy_amb82_mini) x 1
- [SparkFun 6oF IMU Breakout - BMI270](https://www.sparkfun.com/sparkfun-6dof-imu-breakout-bmi270-qwiic.html) x 1
## Example[](#example)
### Introduction[](#introduction)
This example shows how to configure interrupt pins to take measurement.
### Procedure[](#procedure)
Connect the AMB82-mini to I2C_SDA, I2C_SCL, INT1 of the sensor as shown in the diagram below.
[](../../../../_images/image01140.png)
Open the example in File -> Examples -> AmebaWire -> BMI270 -> Interrupts
[](../../../../_images/image02135.png)
Compile and run the example.
The measurements from the sensor will be printed out upon interrupt trigger.
[](../../../../_images/image03114.png)
