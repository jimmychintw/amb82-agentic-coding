Waveshare Epaper
=================

For more source code of epaper refer to [waveshare epaper GitHub](https://github.com/waveshareteam/e-Paper/tree/master/Arduino) and [E-Paper API Analysis
](https://www.waveshare.com/wiki/E-Paper_API_Analysis).

For E-paper Arduino documents refer to [E-paper for Arduino Use](https://www.waveshare.com/wiki/E-paper_for_Arduino_Use).

For hardware connection, it is recommanded to have a [E-Paper Driver HAT](https://www.waveshare.com/wiki/E-Paper_Driver_HAT) to connected between E-paper and Ameba EVBs.

Current version supported
--------------------------

- epd2in9
- epd2in9b
- epd2in9v2
- epd4in2
- epd7in5
    - only compatiable with E-Paper Driver HAT V2.1
